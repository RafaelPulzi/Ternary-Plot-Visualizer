**Visualizador interativo de gráficos ternários** – uma ferramenta front-end para plotar dados em coordenadas ternárias. Permite adicionar pontos, visualizar composições e limpar o gráfico dinamicamente.

## ✨ Funcionalidades

- 📊 **Plotagem de pontos** em coordenadas ternárias (A, B, C).
- ➕ **Adição e remoção** de dados de forma interativa.
- 🧹 **Botão "Clear Data"** para reiniciar o gráfico rapidamente.
- 🎨 Interface visual limpa e responsiva.

## 🚀 Como usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/RafaelPulzi/Ternary-Plot-Visualizer.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Insira os valores para os vértices A, B e C e clique em "Adicionar" para plotar o ponto.
4. Utilize o botão "Clear Data" para limpar todos os pontos do gráfico.
